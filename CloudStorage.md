## 🛠️ ТЕХНОЛОГИЧЕСКИЙ СТЕК

### Backend
- **Node.js** + **Express.js** - сервер
- **MongoDB** + **Mongoose** - база данных
- **JWT** + **bcrypt** - аутентификация
- **Multer** - загрузка файлов
- **Helmet**, **Compression**, **Morgan** - безопасность и оптимизация

### Frontend
- **HTML5**, **CSS3**, **JavaScript (ES6+)**
- **Fetch API** - работа с сервером
- **LocalStorage** - сохранение настроек
- **Адаптивный дизайн** - поддержка всех устройств
- **Темная/светлая тема** - комфортный просмотр
